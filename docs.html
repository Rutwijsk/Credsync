CredSync API Reference

Welcome to the CredSync developer hub. Our unified API allows you to access deep-tier supply chain data, originate credit lines, and manage liquidity pools programmatically.

Get API Keys

Learn about our HMAC signature scheme.

Smart Contracts

View contract addresses and ABIs.

Authentication

CredSync uses API keys to allow access to the API. You can register a new API key at our developer portal.

Terminal 
curl https://api.credsync.in/v1/invoices \
  -H "Authorization: Bearer sk_live_8392..." \
  -H "Content-Type: application/json"

Data APIs

Access raw supply chain data. This endpoint retrieves verified invoices associated with a specific entity (Buyer/Supplier).

GET /v1/entities/{id}/invoices
Response (200 OK) 
{
  "data": [
    {
      "id": "inv_99283",
      "amount": 45000.00,
      "currency": "USD",
      "status": "verified",
      "attestation_hash": "0x7d...a922",
      "risk_score": 92
    }
  ]
}

Credit Workflow

Create liquidity pools or submit invoices for factoring.

POST /v1/pools/create 
// Request Body
{
  "pool_name": "Blue Chip Suppliers Tranche A",
  "asset_type": "invoice_receivable",
  "min_credit_score": 700,
  "target_apy": 12.5
}
